# V1.5 - Narrative Tool Enhancement

## 🎯 Objectif
Transformer l'éditeur technique en véritable outil narratif pour auteurs jeunesse non-techniques.

---

## ✨ Nouvelles fonctionnalités

### 1. Story Engine - Création de livre complet

**Localisation**: Bouton "Créer un livre complet" dans la vue Projets (visible uniquement si un projet est ouvert)

**Fonctionnalités**:
- Interface wizard avec 5 champs obligatoires:
  - Thème de l'histoire
  - Personnage principal
  - Problème rencontré
  - Morale (optionnel)
  - Nombre de pages cible (8, 12, 16, 20, 24, 32)

**Processus automatisé**:
1. Génération d'une histoire complète structurée via GPT-4
2. Extraction automatique du résumé
3. Extraction automatique des personnages
4. Découpage intelligent en pages
5. Création de pages structurées avec templates appropriés
6. Remplissage automatique du texte et des prompts d'illustration

**Résultat**: Livre complet prêt à être illustré et affiné

---

### 2. Templates de pages obligatoires

**Localisation**: Bouton "Nouvelle Page" dans la vue Écriture

**4 templates disponibles**:

#### Illustration pleine page
- Grande zone d'image (90% de la page)
- Texte centré en bas (10%)
- Idéal pour: Pages de titre, scènes importantes

#### Texte court
- Texte centré au milieu de la page
- Grande taille de police (24px)
- Idéal pour: Titres, morales, phrases d'impact

#### Mixte (image + texte)
- Image en haut (60% de la page)
- Texte en bas (40%)
- Idéal pour: Pages narratives standard

#### Double page
- Image plein écran
- Texte superposé avec fond semi-transparent
- Idéal pour: Scènes panoramiques, moments clés

**Avantages**:
- Zones automatiquement créées et positionnées
- Respect des safe areas et bleed
- Cohérence visuelle du livre
- Pas de manipulation technique requise

---

### 3. Chat contextuel projet

**Amélioration majeure**: Le chat IA reçoit maintenant automatiquement:

**Contexte complet envoyé à chaque message**:
- Titre et métadonnées du projet
- Âge cible et type de livre
- **Résumé de l'histoire** (mémoire narrative)
- **Liste complète des personnages** avec descriptions
- **Contenu de toutes les pages existantes**

**Bénéfices**:
- L'IA maintient la cohérence narrative
- Suggestions adaptées au contexte existant
- Pas de répétition ou contradiction
- Assistant narratif intelligent, pas simple chatbot

**Exemple**:
```
Utilisateur: "Ajoute une page où le héros rencontre un ami"
IA: "D'accord ! Puisque ton héros Léo le dragon est timide (page 2), 
je propose qu'il rencontre Sophie la souris qui l'aide à surmonter 
sa peur de voler. Cela s'aligne avec la morale de confiance en soi."
```

---

### 4. Outil "Distribuer du texte"

**Localisation**: Bouton "Distribuer du texte" en haut de la vue Écriture

**Cas d'usage**: L'utilisateur a écrit une longue histoire dans Word/Google Docs

**Processus**:
1. Coller le texte long dans le modal
2. L'IA analyse le texte
3. Découpage intelligent en fonction du nombre de pages existantes
4. Distribution automatique sur chaque page
5. Adaptation du vocabulaire à l'âge cible

**Règles de découpage**:
- 1 à 3 phrases maximum par page
- Coupures narratives naturelles
- Respect de la cohérence de l'histoire
- Vocabulaire adapté à l'âge cible

---

### 5. Résumé du livre auto-généré

**Localisation**: Panneau en haut de la vue Écriture

**Fonctionnalités**:
- Génération automatique via bouton "Générer"
- Édition manuelle possible
- Mise à jour automatique dans le projet
- Utilisé comme mémoire narrative pour l'IA

**Contenu du résumé** (3-4 phrases):
- Personnages principaux
- Problème central
- Résolution
- Morale (si applicable)

**Importance**:
- Sert de référence pour maintenir la cohérence
- L'IA l'utilise dans chaque réponse
- Évite les dérives narratives
- Permet de vérifier la cohérence globale

## 🔧 Modifications techniques

### Nouveaux composants
- `StoryEngineModal.js` - Wizard de génération complète
- `PageTemplateSelector.js` - Sélecteur de templates
- `BookSummaryPanel.js` - Panneau de résumé
- `TextDistributor.js` - Outil de distribution de texte

### Modifications de composants existants
- `ProjectsView.js` - Ajout bouton Story Engine
- `PageList.js` - Intégration sélecteur de templates
- `WritingView.js` - Ajout panneau résumé + bouton distribution
- `AIChat.js` - Contexte projet complet dans chaque message

### Structure de données enrichie

**Projet**:
```javascript
{
  // ... champs existants
  summary: string,           // Résumé narratif
  characters: [{             // Personnages extraits
    id: string,
    name: string,
    description: string,
    age: string,
    appearance: string,
    clothing: string,
    personality: string
  }],
  storyGenerated: boolean,   // Flag génération complète
  generatedAt: string        // Date de génération
}
```

**Page**:
```javascript
{
  // ... champs existants
  template: string,          // Type de template
  imageZones: [{             // Zones d'image prédéfinies
    id: string,
    x: number,
    y: number,
    width: number,
    height: number,
    prompt: string           // Prompt d'illustration
  }],
  illustrationPrompt: string // Prompt global de la page
}
```
